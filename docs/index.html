Why use Squatch?

Haven't Bootstrap and Foundation pretty much got you covered? Yes, both these frameworks offer lots and lots of features, more than a small website needs.

Lightweight

Squatch is designed to give you more freedom and flexibility without overloading you components that need to be customized.

Quantity != Quality

Any responsive website needs certain basic components, navs, buttons, grids, etc. we focus on doing these things well.

Be original

With our flexible and intuitive sass files, we put the design decisions back on the developers and designers where they belong.

Learn more about customizing with Sass >>

Squatch is flexible!

Flexbox grid system makes designing responsive layouts a breeze. Float based grids are no longer needed. Autoprefixer will take care of the differences in modern browsers. IE 9 and below are dead to us.
